Commerce PBZ MyCheckOut
===========

This module implements the Croatian PBZ MyCheckOut payment service [1] for use with Drupal Commerce [2].
You need to configure both the module and the payment service at the website PBZ MyCheckOut.

[1] http://www.pbzcard.hr/hr/online-usluge/prodajna-mjesta/mycheckout
[2] http://drupal.org/project/commerce
